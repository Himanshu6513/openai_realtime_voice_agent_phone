This repository helps you set up an OpenAI Realtime-based Voice AI system integrated with Twilio, enabling phone call–based AI interactions.

Steps to Build and Run the System:

Deploy the repository on a Google Cloud VM instance.

Configure a Twilio phone number that will be used to initiate outbound calls.

Run the application and expose the server using an HTTPS tunnel via ngrok.

Trigger the external API endpoint to initiate the outbound call through Twilio.

This setup allows Twilio calls to connect with the OpenAI Realtime API, enabling a real-time Voice AI agent over phone calls.
